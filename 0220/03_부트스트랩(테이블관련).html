class="table" : 테이블 기본서식

class="table-striped" : 행마다 다른 색상을 적용.

class="table-bordered" : 테이블 셀 테두리 만들기

class="table-hover" : 마우스 오버시 배경색 변경

class="table-dark" : 테이블 제목줄(thead)에 배경색 어둡게 적용

class="table-success", class="table-danger", class="table-info", class="table-primary" : tr, td에 각각 분류색상을 적용할 수 있음.

class="table-responsive" : 화면의 너비가 768px미만일 경우 가로로 스크롤됨.

페이지네이션 - pagenation

nav, ul, li, a태그로 작성하고 nav태그에 aria-label속성을 작성한다.(페이지내 내비게이션이 여러개 있을 수 있기 때문에 목적을 반영하기 위함)

li태그에 class='page-item'을 적용하고 활성화된 메뉴는 'active', 비활성화된 메뉴는 'disabled' class명 적용

a태그에 class='page-link'를 적용

정렬속성은 ul태그에 class='justify-content-center'적용
